# SRE RCA Learning Library

Purpose: Personal RCA learning notes for SRE/AI-SRE growth.

Use this repo to study how incidents were detected, triaged, diagnosed, mitigated, documented, and prevented.

## How to use
1. Read one RCA note at a time.
2. Focus on investigation flow, evidence, mitigation, prevention.
3. Add your own learnings after each incident.
4. Use the interview story section for SRE interview preparation.
5. Update `RCA-Patterns-Learned.md` after every 5-10 incidents.
