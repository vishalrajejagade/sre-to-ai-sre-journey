# Azure VMSS Project 🚀

## Overview
This project demonstrates a scalable cloud architecture using Azure VMs, Load Balancer, and VM Scale Sets with autoscaling.

## Architecture
User → Load Balancer → VMSS → Multiple VMs (NGINX)

## Features
- Load Balancing
- Auto Scaling
- High Availability
- Monitoring and Alerts
- Secure SSH

## Technologies
- Azure VM
- Load Balancer
- VMSS
- NSG
- NGINX

## Learning Outcome
Implemented complete production-like architecture with autoscaling and monitoring.
